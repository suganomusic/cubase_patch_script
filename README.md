## Install
for Mac (MacOS 10.12.6 and Cubase 10)
1. script copy to `{HOME}/Library/Preferences/Cubase 10/Scripts/Patchnames/inactive`.
1. Cubase menu 'Studio' - 'MIDI Device manager'.
1. select Script and MIDI port.
1. Done! :)
